# UID Portfolio

## Project Overview
This project is a **personal portfolio website** built using **HTML and CSS**. It showcases:
- A central landing page
- An introduction/about page
- Hackathon experiences
- Projects

The portfolio is designed to highlight achievements, hackathon participation, and project work in a clean, user-friendly interface.

## Setup Instructions
To set up the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nishthajain7/UID_Portfolio.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd UID_Portfolio
   ```

3. **Check the file structure**
   - `index.html` → Main landing page
   - `about.html`, `hackathons.html`, `projects.html` → Individual sections
   - `styles.css` → Global stylesheet
   - Section-specific CSS files (`about.css`, `hackathons.css`, `projects.css`)
   - `assets/` → Images and other resources

No additional dependencies or package managers are required since this is a static HTML/CSS project.

## How to Run the Project
You can run the portfolio in two simple ways:

### Option 1: Open in Browser
- Double-click `index.html` (or right-click → *Open with Browser*).
- Navigate between sections using the links provided.

### Option 2: Go to the deployed site
[Click here ](https://nishthajain7.github.io/UID_Portfolio/) to visit the portfolio site deployed using github pages
